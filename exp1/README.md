# Experiment 01: Implementation of Basic Logic Circuits using Logisim

## Objective
To understand and implement basic digital logic circuits using Logisim Evolution, including multiplexers, priority encoders, and combinational circuits.

## Background Study
Digital logic circuits operate on binary values (0 and 1).

Basic gates:
- AND: Output is 1 only if both inputs are 1
- OR: Output is 1 if at least one input is 1
- NOT: Inverts the input

These gates are used to build complex circuits like:

- Multiplexer (MUX): Selects one input using select lines
- Priority Encoder: Converts multiple inputs into binary output based on priority

Logisim Evolution is used to design and simulate these circuits.

## Experiment Description
In this experiment:
- An 8:1 Multiplexer was implemented
- A hierarchical multiplexer design was created
- An 8:3 Priority Encoder was designed
- Basic combinational circuits using OR gates were tested
